# 📄 Job Application Form (HTML + CSS)

This is a simple and responsive **Job Application Form** built using **HTML and CSS**. The form includes a hover effect where the form becomes visible when the user hovers over it.

---

## 🚀 Features

* 📱 Responsive design (works on mobile, tablet, desktop)
* 🎨 Clean and simple UI
* ✨ Hover effect (form appears on hover)
* 🧾 Multiple input fields (name, designation, etc.)
* 💡 Beginner-friendly project

---

## 🛠️ Technologies Used

* HTML5
* CSS3
